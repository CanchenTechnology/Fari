如何请求删除账号

方式一：在应用内删除

  1. 打开 Fari 应用并登录你的账号。
  2. 进入账号或个人中心页面。
  3. 点击“删除账号”。
  4. 阅读确认弹窗，点击“永久删除”完成请求。

删除账号是不可恢复操作。完成后，你将无法再使用该账号登录，也无法恢复已删除的云端资料。

方式二：通过邮件请求

如果你无法登录应用，请发送邮件至 dev@medrixai.com，主题建议填写“Fari account deletion request”。

为保护账号安全，请在邮件中提供你希望删除的账号邮箱、登录方式、应用内昵称或其他可帮助我们验证账号归属的信息。请不要发送密码、验证码、身份证件、银行卡或其他不必要的敏感信息。

会删除哪些数据

数据类型 删除说明
账号与认证数据 删除 Firebase Authentication 账号，以及用户资料文档中的昵称、邮箱、头像、生日、城市、时区、会员状态等账号字段。
公开资料 删除用于好友搜索和公开展示的 public profile 数据。
占卜、对话、记忆与偏好 删除该账号拥有的占卜记录、每日神谕、对话历史、记忆、偏好设置和相关子集合数据。
好友与社交数据 删除好友请求、好友关系、拉黑记录、关系占卜参与记录和虚拟好友相关数据。
反馈与客服数据 删除与该账号 UID 关联的反馈记录和客服处理镜像数据。
订阅校验与用量数据 删除与该账号 UID 关联的 IAP 收据校验、支付事件、用量限制等记录，除非需要依法或为争议处理继续保留。

可能保留的数据

在以下情况下，部分数据可能会在合理期限内继续保留：

  • 为满足法律、税务、审计、欺诈防范或平台合规要求。
  • 为处理订阅、退款、拒付、支付争议或应用商店审核请求。
  • 数据已经匿名化或汇总化，无法再合理识别到具体用户。
  • 备份系统中残留的数据会在备份轮换周期内自动清理。

我们不会在账号删除后继续使用可识别账号的数据来提供个性化功能。

处理时间

应用内删除请求通常会立即开始处理。通过邮件提交的删除请求，我们会在收到并完成必要账号验证后尽快处理；复杂请求、支付争议或法律合规场景可能需要更长时间。

If you submit a request by email, we may ask for additional information to confirm that you control the account before deleting data.
