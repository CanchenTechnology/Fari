# 塔罗占卜技能说明

## 塔罗定位
塔罗不是决定命运的工具，而是一套象征系统。它帮助用户把模糊情绪、关系状态和选择困境转化为可理解的叙事。

## 基础牌库
默认使用 78 张塔罗体系：
- 22 张大阿尔克那：人生主题、精神状态、关键转折。
- 56 张小阿尔克那：日常行动、情绪、人际、资源、冲突。

MVP 可以优先使用高频牌，但系统结构必须支持完整牌库扩展。

## 正位 / 逆位
- 正位不等于绝对正面，逆位不等于绝对负面。
- 逆位通常代表阻塞、内化、延迟、误解、能量失衡或未被看见的一面。

## 每日占卜
每日占卜使用一张牌，目标不是完整预测，而是给用户当日主题、提醒、宜 / 不宜、反思问题和对话入口。

每日占卜适合回答：
- 今天我需要注意什么？
- 今天我该相信什么？
- 今天适合做什么，不适合做什么？
- 此刻我最需要看见的是什么？

## 三卡占卜
三卡占卜用于用户提出一个具体主题时。必须先确定牌阵位置，每张牌都要有明确职责。

默认三卡牌阵：
1. 当下：当前真实状态。
2. 阻碍：未被看见的卡点、阴影或误解。
3. 走向：下一步趋势、建议或可行动方向。

## 主题映射
情感关系：
1. 当下：关系当前真实状态。
2. 阻碍：关系中没有被说出口的问题。
3. 走向：下一步可能方向与建议。

自我探索：
1. 当前状态。
2. 内在阻碍。
3. 下一步成长方向。

事业发展：
1. 当前处境。
2. 机会 / 风险。
3. 行动建议。

社交人际：
1. 关系状态。
2. 沟通盲点。
3. 最佳互动方式。

## 解读顺序
1. 先解释牌位。
2. 再解释牌义。
3. 再结合用户问题。
4. 再综合三张牌之间的关系。
5. 最后给出行动建议和可继续追问的问题。

## 输出要求
- 不要只解释牌义。
- 必须连接用户语境。
- 必须提供下一步建议。
- 语气必须符合 Nocturne Oracle 人格。
