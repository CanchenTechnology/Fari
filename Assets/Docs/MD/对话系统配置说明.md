## 二、Unity 场景配置步骤

### 步骤1：创建用户消息预制体 (UserChatItem)

1. 在场景中创建一个 UI 物体作为预制体模板
2. 添加以下组件：
   - `RectTransform`
   - `LoopListViewItem2` (SuperScrollView 组件)
   - `ChatItem` (自定义脚本)
3. 在 `ChatItem` 中绑定以下 UI 元素：
   - `headImage` (Image) - 用户头像
   - `speakerName` (Text) - 用户名（如"小夜"）
   - `speakerContent` (Text) - 消息内容
4. 将该物体拖入 Prefabs 文件夹，命名为 `UserChatItem`

### 步骤2：创建 AI 回复预制体 (AIChatItem)

1. 在场景中创建一个 UI 物体作为预制体模板
2. 添加以下组件：
   - `RectTransform`
   - `LoopListViewItem2` (SuperScrollView 组件)
   - `MessageItem` (自定义脚本)
3. 在 `MessageItem` 中绑定以下 UI 元素：
   - `headImage` (Image) - AI头像
   - `speakerName` (Text) - AI名称
   - `contentText` (Text) - 回复内容
   - `regenerateVoiceButton` (Button) - "重新生成声音"按钮
   - `option1Button` ~ `option4Button` (Button) - 选项按钮
   - `option1Text` ~ `option4Text` (Text) - 选项按钮文本
4. 布局参考截图：
   - 塔罗师样式（3个选项）：显示 option1~3，隐藏 option4
   - 占星师样式（4个选项）：显示 option1~4
5. 将该物体拖入 Prefabs 文件夹，命名为 `AIChatItem`

### 步骤3：配置聊天列表 (LoopListView2)

1. 在 DialogUI 窗口下创建一个 ScrollView
2. 给 ScrollView 添加 `LoopListView2` 组件
3. 在 Inspector 中配置 `ItemPrefabDataList`：
   - 添加两项：
     - Item 1: mItemPrefab = `UserChatItem` 预制体
     - Item 2: mItemPrefab = `AIChatItem` 预制体
4. 配置 ScrollRect：
   - Content 的 Anchor 设置为 Bottom 或 Top（根据排列方向）
   - 建议排列方式：`ListItemArrangeType.BottomToTop`（新消息在底部）

### 步骤4：配置 DialogUI

1. 找到场景中的 DialogUI 物体
2. 确保 `DialogUI` 脚本组件已挂载
3. 在 Inspector 中配置以下字段：
   - `chatListView` - 拖拽步骤3创建的 LoopListView2
   - `userItemPrefabName` - 输入 `"UserChatItem"`
   - `aiItemPrefabName` - 输入 `"AIChatItem"`
   - `dialogSystem` - 可选，留空会自动创建

### 步骤5：配置 DialogSystem（可选）

1. 在场景中创建一个空物体，命名为 `DialogSystem`
2. 挂载 `DialogSystem` 脚本
3. 可配置以下参数：
   - `CurrentDivinerType` - 默认占卜师类型（塔罗师/占星师）
   - `TarotDivinerName` / `AstrologyDivinerName` - 占卜师名称
   - `UserName` - 用户名
   - `TarotSystemPrompt` / `AstrologySystemPrompt` - AI角色设定提示词

## 三、功能说明

### 已实现功能

1. **发送消息**
   - 在输入框输入内容，点击发送按钮
   - 自动生成用户 ChatItem 显示在列表中
   - 自动清空输入框

2. **AI 回复**
   - 自动调用 DeepSeek API (deepseek-v4-pro)
   - 根据当前占卜师类型使用不同的系统提示词
   - 生成 AI MessageItem 显示在列表中
   - 自动滚动到底部

3. **切换占卜师**
   - 点击"切换占卜师"按钮
   - 在塔罗师和占星师之间切换
   - 切换后AI回复风格和选项按钮会相应变化

4. **选项按钮**
   - 塔罗师回复：3个选项（为这个问题选牌阵 / 继续追问 / 明天再看这条线索）
   - 占星师回复：4个选项（看这段关系的周期 / 分析今日星象 / 看下一周趋势 / 保存明日回看）
   - 点击选项会触发对应功能或作为新问题发送

5. **重新生成声音**
   - 点击AI消息中的"重新生成声音"按钮
   - 可触发语音合成（需自行实现TTS功能）

### DeepSeek API 配置

AI 请求已改为通过 Firebase Cloud Functions 代理：
- 非流式接口：`aiChat`
- 流式接口：`aiChatStream`
- API Key 不再放在 Unity 客户端中，需通过 Firebase Functions Secret `DEEPSEEK_API_KEY` 配置。
- 当前客户端默认模型：`deepseek-chat`

## 四、注意事项

1. 确保 SuperScrollView 插件已正确导入
2. 确保预制体名称与 `DialogUI` 中的 `userItemPrefabName` 和 `aiItemPrefabName` 一致
3. 如果需要修改 API 密钥，请更新 Firebase Functions Secret `DEEPSEEK_API_KEY`
4. 头像加载功能目前为预留接口，需要根据实际项目资源管理方式实现
5. 语音合成功能目前为预留接口，需要接入 TTS 服务
